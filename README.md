# node_ftp
## 环境
1. Mac OS X
2. Nodejs V4.2.2

## 目标
1. 用nodejs的ftp模块，访问ftp服务器。
2. 切换到指定的ftp文件夹
3. 列出目标文件夹下的文件清单

## 代码
### 开启FTP服务
我用的是shell命令：
```bash
# ftp的根目录是系统用户的根目录~
sudo -s launchctl load -w /System/Library/LaunchDaemons/ftp.plist
```
